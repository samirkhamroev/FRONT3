# Приложение "Заметки" (PWA)

Данный репозиторий является результатом выполнения практических занятий №13-18 по дисциплине "Фронтенд и бэкенд разработка". 
Это полноценное Progressive Web Application (PWA) для создания и управления заметками, поддерживающее работу в офлайн-режиме, синхронизацию в реальном времени через WebSockets и Push-уведомления с отложенным запуском (Snooze).

## Технологии и реализованные практики

- **HTML/CSS/JS (Vanilla):** Построен интерфейс, добавлены стили (`chota.css`).
- **Практика 13 (Service Worker):** Реализовано кэширование и офлайн-работа. 
- **Практика 14 (Web App Manifest):** Создан `manifest.json`, добавлены иконки. Приложение устанавливается как самостоятельное.
- **Практика 15 (App Shell):** Динамическая подгрузка контента (`/content/home.html`, `/content/about.html`) с использованием стратегий Cache First (для статики) и Network First (для динамики).
- **Практика 16 (WebSocket + Push):** Подключён `Socket.IO`. При добавлении заметки в браузере, сигнал летит на сервер и он уведомляет об этом других клиентов. Пользователь может включить Push-уведомления (генерация VAPID ключей, подписка `subscribeToPush`).
- **Практика 17 (Детализация Push):** В форму добавлена возможность выбора даты `datetime-local` для срабатывания напоминания. Уведомление содержит кнопку "Отложить на 5 минут", которая через Service Worker вызывает `/snooze`.
- **Практика 18:** Тестирование работоспособности приложения и подготовка этого `README.md`.

## Инструкция по запуску

1. Убедитесь, что установлен Node.js (рекомендуется v20+).
2. Склонируйте репозиторий.
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Запустите сервер:
   ```bash
   node server.js
   ```
5. Откройте в браузере [http://localhost:3001](http://localhost:3001).
